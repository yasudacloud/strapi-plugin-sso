## Strapi plugin strapi-plugin-sso

This plugin can provide single sign-on.

Currently supports Cognito user pool and Google accounts.


### Please prepare the following
- Strapi Version4
- **strapi-plugin-sso**
- Google Account or AWS Cognito UserPool
